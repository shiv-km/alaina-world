🔒 Safety & Privacy

We collect zero personal information. No name, no email, no location, no profile. Nothing leaves your device.

The app stores only game progress (stars, streak, name, avatar) in your browser's localStorage. This data never leaves the device and is never transmitted to any server. You can clear it at any time through your browser settings.

There are no user accounts, no sign-ups, no passwords. Children can play immediately without any registration.

🤖 AI Features (Optional)

AI features are completely optional. All 28 games and all 15 stories work without any API key.

Three features optionally use OpenAI's API: Story Illustrations, AI Story Creator, and Coloring Studio. To use them, a parent or guardian must:

  • Create an OpenAI account at platform.openai.com
  • Generate an API key and paste it into the app's settings
  • Each coloring page or story generation costs approximately $0.04–$0.06 USD (charged to your OpenAI account)

The API key is stored only in the browser's localStorage and sent directly to OpenAI — not to us. You can remove it at any time from the ⚙️ Settings button inside Story Time or the Coloring Studio.

💰 Cost & Monetization

📱 Device & Technical Notes

  • Works on any modern browser (Chrome, Safari, Firefox, Edge)
  • Designed for phone, tablet, and desktop
  • No app store download needed — open directly in a browser
  • No internet required for the 28 games after the initial page load (fonts load from Google)
  • AI features require internet to reach OpenAI's servers
  • Sound uses the Web Audio API — ensure device is not muted
  • Speech features (Talking ABC, Number Bounce, Animal Sounds, Voice Narration) use the browser's built-in speech synthesis — no external service

❓ Frequently Asked Questions

Is this app safe for my child to use unsupervised? Yes. There is no chat, no social features, no external links within the app (except optional OpenAI API calls you set up), and no way for strangers to contact your child.
How do I remove the OpenAI API key? Open Story Time (storybook), tap the "AI Illustrations" banner at the top, then tap "Remove Key". Or open Coloring Studio and tap "⚙️ OpenAI key needed" to access key settings.
How do I reset all progress (stars, streak, etc.)? Open your browser settings, find "Site Data" or "Cookies and Site Data" for this site, and clear it. This removes all localStorage data including stars, streak, and any API keys.
Can my child's custom stories be seen by others? No. Custom stories are stored only in your browser's localStorage on your device. They are not uploaded anywhere and cannot be accessed by anyone else.
What age is this designed for? Primarily ages 2–8, though the Brain Games (math, memory, word match) work well up to age 10. The games are designed to be discoverable and self-explanatory for young children.
Is there a way to limit which games my child can access? Not currently — all games are accessible from the main menu. If this is important to you, please open a request on our GitHub page.
